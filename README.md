# booking-form-website-clean

Clean GitHub Pages version of the Ferdie Nervida inquiry site for:

- speaking inquiries
- training inquiries
- crypto recovery case evaluation
- general service inquiries

## Local preview

```bash
cd "/Users/fjnervida/Documents/New project/booking-form-website-clean"
python -m http.server 8083
```

Open:

- `http://127.0.0.1:8083/`

## Form handling

The form submits through FormSubmit using:

- `https://formsubmit.co/ajax/contact@ferdienervida.com`

Before using it live, make sure the destination email has been activated with FormSubmit.

## Deployment

This project is intended for GitHub Pages with the custom domain:

- `book.ferdienervida.com`
